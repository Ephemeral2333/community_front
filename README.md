# HAUT_Community

#### 介绍

软件工程项目实践 -- **Gathering Circle** 社交平台--前端前台代码

实现了一个校园社区交流平台，前端前台使用Vue 2.0 + JavaScript开发

#### 技术栈

使用到了Vue 2.0 + JavaScript + Element UI + Buefy 等
